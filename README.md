**TUBM** is an open-source web application developed by **Victor Fioravante Xavier**.  
It serves as a modular base project combining **Node.js**, **Express**, **EJS**, and **Webpack**, optimized for clean structure, scalability, and deployment via **Vercel**.

Live demo: **https://tubm.vercel.app**

---

## üöÄ Overview

TUBM was designed to provide a solid foundation for applications that require:

- Server-side rendering using **EJS**
- Clean, modular route organization
- Efficient asset bundling with **Webpack**
- Maintainable frontend structure
- Easy deployment and CI/CD workflow
- Fast prototyping and MVP development

## üíª Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/VictorFioravanteXavier/TUBM.git
cd TUBM 
```

### 2. Install dependencies
```bash
npm install
```

### 3. Configure environment variables 
melhorar...
```bash
Create a .env file:

PORT=3000
DATABASE_URL=your_database_here
```

### 4. Start the development server
```bash
npm run dev
```

## Production:
```bash
npm start


Access your local instance at:
http://localhost:3000
```
